### Installation Guide

#### STEP 1

clone repo `'git clone https://github.com/kelanik8/track-management.git'`

### FRONT-END

##### STEP 1
dir `'cd frontend'`
install packages`'yarn install'`

##### STEP 2

run development `'yarn dev'`



### BACK-END

##### STEP 1
dir `'cd backend'`
install packages`'yarn install'`

##### STEP 2

run watch `'yarn watch'`

##### STEP 3

run development `'yarn dev'`
